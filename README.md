Snazzl is a **Quick-Commerce Fashion Delivery Platform** that delivers fashion and accessories within **30 minutes**.

Built using **Flutter**, the platform supports customers, shop owners, delivery partners, and administrators through dedicated applications and portals.

The system is optimized to handle **10,000+ users in its initial phase**, with a scalable backend architecture designed for future expansion.
